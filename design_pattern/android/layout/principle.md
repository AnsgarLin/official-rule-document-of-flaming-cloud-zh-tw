# 原則
Ｍaterial Design運用實體印刷的概念，來建立物件之間的關聯和意義。同時也繼承印刷設計的技巧，例如網格、模板和一致的環境設定。從而建立一個可適用於各種大小的佈局。

## 紙張概念
由於是繼承實體印刷的概念，所以物件具有紙張的特性，例如平面色彩、不透明的字體。

關於物件詳細的特性，請查看[概念](../concept.html)章節。
