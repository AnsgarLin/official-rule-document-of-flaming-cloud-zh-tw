# 圖示
Google 官方所使用的圖示已經發展出一套完整且靈活的單一系統，並且具有固定的規則可遵循。依照用途，圖示大致可分為以下兩類：

* 產品圖示
* 系統圖示

## 產品圖示
為產品標誌，用於商城。

### 圖層
圖示設計可以分成數個圖層，通過相同的圖層設計，可以使各種產品圖示具有相同的象徵語言。熟悉這些圖層可以幫助了解每種產品圖示間的差異和結構，也可以理解如何用正確的圖示進行表達。
1. 完成圖
2. 背景
3. 前景
4. 色塊
5. 陰影
